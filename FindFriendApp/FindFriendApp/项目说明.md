                                                   [1.项目概述]：


                                                    本项目是基于 Java 主语言 + C++ 底层运算核心 
                                                        开发的安卓设备交友类应用（毛坯交付版）
                                                                适配 Android 12+ 系统
                       核心亮点：使用 NDK/CMake 实现 C++ 底层定位计算、距离测算，性能优于纯 Java 实现，可直接二次开发迭代完整功能
                                                   [2.技术栈]：


                                                   前端开发：Java + Android XML 布局
                                                   底层核心：C++、NDK、CMake
                                                   构建工具：Gradle 9.4
                                                   运行环境：Android SDK 34、AndroidX
                                                   系统支持：Arch Linux（编译环境）、Android 12+（运行环境）
