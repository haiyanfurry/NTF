# FindFriendApp

FindFriendApp 是一个多平台应用系统，支持 iOS、Debian、RedHat、Windows 和 Android 平台，为用户提供定位、社交、计划管理、入场码生成、内部交易审核、照片传输等功能。

## 功能特性

- **定位功能**：查看同伴的实时定位（需对方开启定位）
- **社交功能**：支持与他人扩列，建立联系
- **计划/展商管理**：查看计划和展商信息，支持购票
- **入场码功能**：生成购票对应的入场码
- **内部交易审核**：支持内部售卖稿件、门票等物资
- **现场照片传输**：支持快速传输大量照片（单次最多250张）
- **多媒体信息传输**：支持传输照片、视频、音频等素材
- **信息查看**：查看其他用户的公开信息及详情
- **身份区分标识**：区分账号属性（是否有装、是否扁禾、是否为官方）
- **消息管理**：支持免打扰设置，仅官方信息可读
- **IP 快速切换**：支持在特定场景（如漫展）下便捷切换服务器 IP
- **同城用户发现**：能够识别并连接使用相同软件的主办方设备

## 技术架构

- **客户端-服务器架构**：各平台客户端与服务器端通过网络通信
- **模块化设计**：每个平台采用模块化设计，确保代码可维护性和可扩展性
- **本地数据库**：使用 SQLite 进行本地数据存储和用户认证
- **跨平台通信**：支持不同平台间的设备发现和数据传输

## 安装与使用

### iOS 平台
1. 打开 App Store
2. 搜索 "FindFriendApp"
3. 点击 "获取" 按钮
4. 等待下载和安装完成
5. 点击应用图标启动

### Debian 平台
1. 进入 Debian 目录
2. 运行 `make` 命令
3. 运行 `./FindFriendApp` 启动应用

### RedHat 平台
1. 进入 RedHat 目录
2. 创建 build 目录：`mkdir build && cd build`
3. 运行 CMake：`cmake ..`
4. 构建项目：`make`
5. 运行应用：`./FindFriendApp`

### Windows 平台
1. 打开 Visual Studio 项目
2. 选择 Release 配置
3. 构建并运行

### Android 平台
1. 打开 Android Studio 项目
2. 选择目标设备
3. 构建并运行

- **命名规范**：类名使用 PascalCase，方法名和变量名使用 camelCase，常量使用 UPPER_SNAKE_CASE
