# FindFriendApp 开发文档

## 项目概述

FindFriendApp 是一个多平台应用系统，支持 iOS、Debian、RedHat、Windows 和 Android 平台，提供用户定位、社交、计划管理、入场码生成、内部交易审核、照片传输等功能。

## 技术架构

### 整体架构
- **客户端-服务器架构**：各平台客户端与服务器端通过网络通信
- **模块化设计**：每个平台采用模块化设计，确保代码可维护性和可扩展性
- **本地数据库**：使用 SQLite 进行本地数据存储和用户认证
- **跨平台通信**：支持不同平台间的设备发现和数据传输

### 各平台技术栈

| 平台 | 主要语言 | UI 框架 | 网络库 | 数据库 |
|------|---------|--------|--------|--------|
| iOS | Swift | UIKit | 原生网络 | SQLite |
| Debian | C | GTK | 原生网络 | SQLite |
| RedHat | C++ | Qt | 原生网络 | SQLite |
| Windows | C++ | Qt | 原生网络 | SQLite |
| Android | C++/Java | Android SDK | 原生网络 | SQLite |

## 核心功能模块

### 1. 用户认证与管理
- **登录/注册**：支持用户注册和登录，用户名唯一性检查
- **用户信息管理**：存储和管理用户信息，包括身份标识
- **本地数据库**：使用 SQLite 进行本地数据存储

### 2. 网络管理
- **IP 快速切换**：支持在特定场景下便捷切换服务器 IP
- **本地设备发现**：使用 UDP 广播实现同城用户发现
- **服务器连接**：管理与服务器的连接状态

### 3. 多媒体传输
- **照片传输**：支持单次最多 250 张照片的快速传输
- **文件传输**：支持多种类型文件的传输
- **无提交式传输**：实现自动传输和保存

### 4. 定位功能
- **实时定位**：在对方开启定位的前提下，查看同伴的实时定位
- **时空功能**：叠加时空信息，提供更丰富的定位体验

### 5. 社交功能
- **扩列**：支持与他人建立联系
- **消息管理**：支持免打扰设置，仅官方信息可读
- **身份标识**：区分账号属性（是否有装、是否扁禾、是否为官方）

### 6. 计划/展商管理
- **计划查看**：查看当前或已公布的计划
- **展商管理**：查看展商进度及见票情况
- **购票业务**：针对未合作展商，开通购票业务

### 7. 入场码功能
- **生成入场码**：完成前置流程并预留权限后，生成购票对应的入场码

### 8. 内部交易审核
- **交易管理**：支持内部售卖稿件、门票等物资
- **人工审核**：交易需经人工审核后，打入驻点

## 各平台实现细节

### iOS 平台
- **优化目标**：启动时间 < 2 秒，内存占用降低 20%，响应速度提升 30%
- **稳定性**：崩溃率控制在 0.1% 以下
- **兼容性**：支持 iOS 12 及以上版本
- **实现**：使用 Swift 语言，采用模块化设计，优化资源加载和内存管理

### Debian 平台
- **IP 快速切换**：实现便捷的服务器 IP 切换功能
- **本地设备发现**：使用 UDP 广播实现同城用户发现
- **GUI**：使用 GTK 开发图形化操作界面
- **实现**：使用 C 语言，模块化设计，支持实时网络状态监控

### RedHat 平台
- **专用 GUI**：开发 RedHat 系统专用图形化操作界面
- **IP 管理**：集成 IP 快速切换功能
- **设备发现**：支持本地设备发现机制
- **实现**：使用 C++ 语言，Qt 框架，模块化设计

### Windows 平台
- **商家专用客户端**：开发 Windows PC 商家客户端
- **无缝对接**：与现有 Debian/RedHat 服务端无缝对接
- **功能完整**：满足商业场景下的操作需求
- **实现**：使用 C++ 语言，Qt 框架，模块化设计

### Android 平台
- **C++ 开发**：使用 C++ 语言开发核心功能
- **跨平台调试**：支持在 Windows 和 Linux 环境下进行快速调试与二次开发
- **性能与稳定性**：达到同类应用的行业标准
- **实现**：使用 JNI 实现 C++ 与 Java 的集成，支持安卓 8.0 及以上版本

## 构建与部署

### iOS 平台
1. 打开 Xcode 项目
2. 选择目标设备
3. 构建并运行

### Debian 平台
1. 进入项目目录
2. 运行 `make` 命令
3. 运行 `./FindFriendApp` 启动应用

### RedHat 平台
1. 进入项目目录
2. 创建 build 目录：`mkdir build && cd build`
3. 运行 CMake：`cmake ..`
4. 构建项目：`make`
5. 运行应用：`./FindFriendApp`

### Windows 平台
1. 打开 Visual Studio 项目
2. 选择 Release 配置
3. 构建并运行

### Android 平台
1. 打开 Android Studio 项目
2. 选择目标设备
3. 构建并运行
