# FindFriendApp 用户手册

## 1. 应用介绍

FindFriendApp 是一个多平台应用系统，支持 iOS、Debian、RedHat、Windows 和 Android 平台，为用户提供定位、社交、计划管理、入场码生成、内部交易审核、照片传输等功能。

### 主要功能
- **定位功能**：查看同伴的实时定位（需对方开启定位）
- **社交功能**：支持与他人扩列，建立联系
- **计划/展商管理**：查看计划和展商信息，支持购票
- **入场码功能**：生成购票对应的入场码
- **内部交易审核**：支持内部售卖稿件、门票等物资
- **现场照片传输**：支持快速传输大量照片
- **多媒体信息传输**：支持传输照片、视频、音频等素材
- **信息查看**：查看其他用户的公开信息及详情
- **身份区分标识**：区分账号属性（是否有装、是否扁禾、是否为官方）
- **消息管理**：支持免打扰设置，仅官方信息可读

## 2. 安装指南

### iOS 平台
1. 打开 App Store
2. 搜索 "FindFriendApp"
3. 点击 "获取" 按钮
4. 等待下载和安装完成
5. 点击应用图标启动

### Debian 平台
1. 下载 FindFriendApp-Debian.tar.gz 文件
2. 解压文件：`tar -zxvf FindFriendApp-Debian.tar.gz`
3. 进入解压目录：`cd FindFriendApp_PC_Debian`
4. 运行应用：`./FindFriendApp`

### RedHat 平台
1. 下载 FindFriendApp-RedHat.tar.gz 文件
2. 解压文件：`tar -zxvf FindFriendApp-RedHat.tar.gz`
3. 进入解压目录：`cd FindFriendApp_PC_RedHat`
4. 运行应用：`./FindFriendApp`

### Windows 平台
1. 下载 FindFriendApp-Windows.exe 安装文件
2. 双击安装文件
3. 按照安装向导完成安装
4. 点击桌面快捷方式启动应用

### Android 平台
1. 下载 FindFriendApp-Android.apk 文件
2. 打开文件管理器，找到下载的 APK 文件
3. 点击安装
4. 安装完成后，点击应用图标启动

## 3. 首次使用

### 注册账号
1. 打开应用
2. 点击 "注册" 按钮
3. 输入用户名和密码
4. 点击 "注册" 按钮
5. 注册成功后自动登录

### 登录账号
1. 打开应用
2. 输入用户名和密码
3. 点击 "登录" 按钮
4. 登录成功后进入主界面

### 基本设置
1. 登录后，点击 "设置" 图标
2. 配置服务器 IP（默认使用本地服务器）
3. 设置个人信息
4. 开启或关闭定位服务

## 4. 功能使用指南

### 4.1 定位功能
1. 点击 "定位" 选项卡
2. 开启定位服务（首次使用需授权）
3. 查看地图上的同伴位置
4. 点击同伴图标查看详细信息

### 4.2 社交功能
1. 点击 "社交" 选项卡
2. 点击 "扩列" 按钮，搜索或扫码添加好友
3. 点击好友列表中的好友，进入聊天界面
4. 点击 "消息设置"，开启或关闭免打扰

### 4.3 计划/展商管理
1. 点击 "计划" 选项卡
2. 查看当前或已公布的计划
3. 点击 "展商" 选项，查看展商信息和进度
4. 点击 "购票" 按钮，购买未合作展商的门票

### 4.4 入场码功能
1. 完成前置流程并预留权限
2. 点击 "入场码" 选项卡
3. 点击 "生成入场码" 按钮
4. 出示入场码进行验证

### 4.5 内部交易审核
1. 点击 "交易" 选项卡
2. 点击 "创建交易" 按钮，填写交易信息
3. 等待人工审核
4. 审核通过后，交易打入驻点

### 4.6 照片传输
1. 点击 "传输" 选项卡
2. 点击 "选择照片" 按钮，最多选择 250 张照片
3. 选择接收设备
4. 点击 "发送" 按钮，自动传输照片

### 4.7 多媒体信息传输
1. 点击 "传输" 选项卡
2. 点击 "选择文件" 按钮，选择照片、视频、音频等素材
3. 选择接收设备
4. 点击 "发送" 按钮，快速传输文件

### 4.8 信息查看
1. 点击 "用户" 选项卡
2. 搜索或浏览其他用户
3. 点击用户头像，查看公开信息及详情

### 4.9 身份区分标识
1. 点击 "我的" 选项卡
2. 查看个人身份标识
3. 官方账号会显示特殊标识

### 4.10 消息管理
1. 点击 "消息" 选项卡
2. 点击 "设置" 图标
3. 开启或关闭免打扰
4. 查看官方消息和群组消息

## 5. 高级功能

### 5.1 IP 快速切换
1. 点击 "设置" 选项卡
2. 点击 "网络设置"
3. 输入新的服务器 IP 地址
4. 点击 "保存" 按钮
5. 应用会自动连接到新的服务器

### 5.2 同城用户发现
1. 点击 "发现" 选项卡
2. 开启 "本地发现" 功能
3. 应用会自动搜索同城使用相同软件的设备
4. 点击发现的设备，建立连接

### 5.3 跨平台调试（仅开发人员）
1. 连接设备到电脑
2. 开启 USB 调试模式
3. 使用 Android Studio 或其他开发工具进行调试

## 6. 常见问题解答

### 6.1 登录问题
**Q: 忘记密码怎么办？**
A: 点击登录界面的 "忘记密码" 链接，按照提示重置密码。

**Q: 用户名已存在怎么办？**
A: 系统会自动为重复的用户名添加编号，例如 "user1"、"user2" 等。

### 6.2 定位问题
**Q: 定位不准确怎么办？**
A: 确保开启了定位服务，并在开阔地带使用。

**Q: 看不到同伴位置怎么办？**
A: 确保同伴也开启了定位服务，并且你们是好友关系。

### 6.3 传输问题
**Q: 照片传输失败怎么办？**
A: 检查网络连接，确保双方设备在同一网络环境下。

**Q: 传输速度慢怎么办？**
A: 确保网络连接稳定，避免同时传输大量文件。

### 6.4 服务器问题
**Q: 连接服务器失败怎么办？**
A: 检查网络连接，或尝试切换服务器 IP。

**Q: 服务器 IP 在哪里设置？**
A: 在 "设置" -> "网络设置" 中设置服务器 IP。

### 6.5 其他问题
**Q: 应用崩溃怎么办？**
A: 重启应用，如果问题持续，联系客服。

**Q: 如何更新应用？**
A: 在应用商店或官方网站下载最新版本。
